Projet e-commerce de Kreuwen Julie et Laplace Danny

Pour lancer le projet et lier la data base :
 -> cloner ou mettre le repo dans le dossier "C:\xampp\htdocs" de votre PC
 -> lancer l'application XAMPP (+ start "Apache" et "MySQL") puis cliquer sur Admin de "MySQL"
 -> une fois sur "http://localhost/phpmyadmin", créer une nouvelle data base nommée "climbingzonedb"
    ensuite importer le fichier suivant : "ClimbingZoneDB.sql" (Ignorer les erreurs s'il y en a)
 => Votre BD est créé et configuré

 -> plus qu'à lancer le fichier "index.php" dans votre navigateur depuis votre IDE.
